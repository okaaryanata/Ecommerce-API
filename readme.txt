Requirment : 
- node js
- npm
- mongodb
- express
- jsonwebtoken
- mongoose
- body-parser
- nodedemon
- pastikan port 5000 tidak digunakan
- text editor (vscode, atom atau apapun)
- postman (untuk test API)

Menjalankan aplikasi
- jalankan mongodb ('sudo mongod' pada terminal)
- jalankan node ('nodemon index.js' pada terminal)
- buka 'localhost:5000/api/test' pada browser atau postman, jika API berjalan maka akan menghasilkan 'REST API RUNNING'
- database schema terdapat pada : folder 'models'
- terdapat beberapa data yang dapat di import ke database, terdapat pada folder 'wir db'
- untuk contoh cURL postman dapat mengakses link berikut : 
https://documenter.getpostman.com/view/7748154/S1ZxapC7
